## Overview

To process a payment with Checkout, you need to create a `CheckoutRequest` with transaction details and call `startCheckoutActivityForResult()` to launch the payment WebView.

## Creating a CheckoutRequest

The `CheckoutRequest` contains all information about the transaction:

```kotlin
val request = CheckoutRequest(
    totalAmount = totalAmount,
    buyer = buyer,
    items = items,
    requestReferenceNumber = requestReferenceNumber,
    redirectUrl = redirectUrl,
    metadata = metadata
)
```

### Required Parameters

#### Total Amount

The transaction amount with currency information:

```kotlin
import com.paymaya.sdk.android.common.models.TotalAmount
import java.math.BigDecimal

val totalAmount = TotalAmount(
    value = BigDecimal("100.00"),
    currency = "PHP"
)
```

<Note>
Use `BigDecimal` for precise monetary calculations. The `value` should represent the total amount in the specified currency.
</Note>

#### Items

A list of items being purchased:

```kotlin
import com.paymaya.sdk.android.checkout.models.Item
import com.paymaya.sdk.android.common.models.TotalAmount
import java.math.BigDecimal

val items = listOf(
    Item(
        name = "Product Name",
        quantity = 1,
        code = "SKU-001",
        description = "Product description",
        totalAmount = TotalAmount(
            value = BigDecimal("100.00"),
            currency = "PHP"
        )
    )
)
```

**Item Properties:**
- `name` (required) - Item's product name
- `totalAmount` (required) - Item's total price
- `quantity` (optional) - Item's quantity
- `code` (optional) - Item's SKU code
- `description` (optional) - Item's description
- `amount` (optional) - Item's unit price with fee breakdown

#### Request Reference Number

A unique identifier for this transaction from your system:

```kotlin
val requestReferenceNumber = "ORDER-${System.currentTimeMillis()}"
```

<Warning>
Ensure each transaction has a unique `requestReferenceNumber` to avoid duplicate processing.
</Warning>

#### Redirect URLs

URLs where the customer will be redirected after payment completion:

```kotlin
import com.paymaya.sdk.android.common.models.RedirectUrl

val redirectUrl = RedirectUrl(
    success = "https://yourapp.com/success",
    failure = "https://yourapp.com/failure",
    cancel = "https://yourapp.com/cancel"
)
```

<Note>
While these URLs are required by the API, the Android SDK handles results through `onActivityResult()` instead of actual redirects. You can use placeholder URLs like `http://success.com`.
</Note>

### Optional Parameters

#### Buyer Information

Optional customer details for the transaction:

```kotlin
import com.paymaya.sdk.android.checkout.models.Buyer
import com.paymaya.sdk.android.checkout.models.Contact
import com.paymaya.sdk.android.checkout.models.Address

val buyer = Buyer(
    firstName = "Juan",
    middleName = "Cruz",
    lastName = "Dela Cruz",
    contact = Contact(
        phone = "+639171234567",
        email = "juan.delacruz@example.com"
    ),
    shippingAddress = Address(
        line1 = "123 Main Street",
        line2 = "Unit 456",
        city = "Manila",
        state = "Metro Manila",
        zipCode = "1000",
        countryCode = "PH"
    ),
    billingAddress = Address(
        line1 = "123 Main Street",
        city = "Manila",
        state = "Metro Manila",
        zipCode = "1000",
        countryCode = "PH"
    )
)
```

<Note>
If you provide buyer information, at least one field must be populated. If the email is provided, a payment receipt will be sent to that address.
</Note>

#### Metadata

Custom data for tracking or processing:

```kotlin
import org.json.JSONObject

val metadata = JSONObject().apply {
    put("userId", "12345")
    put("cartId", "cart-789")
    put("source", "android-app")
}
```

## Starting the Payment Flow

Call `startCheckoutActivityForResult()` to launch the payment WebView:

```kotlin
payMayaCheckoutClient.startCheckoutActivityForResult(this, request)
```

### Method Signature

```kotlin
fun startCheckoutActivityForResult(
    activity: Activity,
    request: CheckoutRequest
)
```

**Parameters:**
- `activity` - Current Activity context
- `request` - The `CheckoutRequest` containing payment details

## Complete Example

```kotlin
import android.app.Activity
import android.os.Bundle
import com.paymaya.sdk.android.checkout.PayMayaCheckout
import com.paymaya.sdk.android.checkout.models.*
import com.paymaya.sdk.android.common.LogLevel
import com.paymaya.sdk.android.common.PayMayaEnvironment
import com.paymaya.sdk.android.common.models.RedirectUrl
import com.paymaya.sdk.android.common.models.TotalAmount
import org.json.JSONObject
import java.math.BigDecimal

class CheckoutActivity : Activity() {

    private val payMayaCheckoutClient = PayMayaCheckout.newBuilder()
        .clientPublicKey("pk-NCLk7JeDbX1m22ZRMDYO9bEPowNWT5J4aNIKIbcTy2a")
        .environment(PayMayaEnvironment.SANDBOX)
        .logLevel(LogLevel.WARN)
        .build()

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        // Setup UI...
    }

    private fun startPayment() {
        // Create total amount
        val totalAmount = TotalAmount(
            value = BigDecimal("100.00"),
            currency = "PHP"
        )

        // Create items
        val items = listOf(
            Item(
                name = "Sample Product",
                quantity = 1,
                code = "PROD-001",
                description = "A sample product",
                totalAmount = TotalAmount(
                    value = BigDecimal("100.00"),
                    currency = "PHP"
                )
            )
        )

        // Create buyer info (optional)
        val buyer = Buyer(
            firstName = "Juan",
            lastName = "Dela Cruz",
            contact = Contact(
                phone = "+639171234567",
                email = "juan@example.com"
            )
        )

        // Create redirect URLs
        val redirectUrl = RedirectUrl(
            success = "http://success.com",
            failure = "http://failure.com",
            cancel = "http://cancel.com"
        )

        // Create metadata (optional)
        val metadata = JSONObject().apply {
            put("orderId", "12345")
        }

        // Create checkout request
        val request = CheckoutRequest(
            totalAmount = totalAmount,
            buyer = buyer,
            items = items,
            requestReferenceNumber = "REF-${System.currentTimeMillis()}",
            redirectUrl = redirectUrl,
            metadata = metadata
        )

        // Start checkout
        payMayaCheckoutClient.startCheckoutActivityForResult(this, request)
    }
}
```

## What Happens Next

When you call `startCheckoutActivityForResult()`:

<Steps>
  <Step title="WebView Launches">
    The SDK launches a new Activity with a WebView showing the PayMaya checkout page
  </Step>
  <Step title="Customer Completes Payment">
    The customer enters payment details and completes the transaction on the PayMaya-hosted page
  </Step>
  <Step title="Result Returned">
    The payment result is returned to your Activity's `onActivityResult()` method
  </Step>
</Steps>

<Note>
If the user presses the Back button to close the checkout Activity, the SDK automatically checks the payment status (if the `checkoutId` was retrieved) and maps it to the appropriate result.
</Note>
